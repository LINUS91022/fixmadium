Why does antivirus say "Trojan.Boxter"?

This is a false positive — a mistake made by antivirus engines, not evidence of actual malware. Trojan.Boxter is a generic, heuristic detection label used when a file matches a behavioral pattern, not an actual known virus signature.

This .bat file does several things that look suspicious to automated scanners: it requests admin rights, uses PowerShell commands, kills processes with taskkill, deletes folders, and downloads .exe files from the internet. Each of those actions individually is normal. Together in a batch script, heuristic engines flag it because malware does the same things.

The difference is intent. This tool does those things to clean and reinstall Madium, not to harm your system. The source code is fully readable — open the .bat in Notepad and you can verify every single line yourself before running it.

You can also check the file hash on VirusTotal. Most engines will clear it — only generic heuristic detectors flag it, and they do so because of the script's behavior, not because it contains any actual malicious payload.
